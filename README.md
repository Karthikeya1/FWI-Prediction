# FWI-Prediction
Predicts Fire Weather Index using given data.
This Problem is solved using Ridge Regression.
Linear, Lasso, Ridge, ElasticNet were used.
But Ridge has the best accuracy of 98%.
Tech Stack:
  Python
  Flask
  Numpy
  Pandas
  Scikit-Learn

Run Project:
  pip install -r requirements.py
  python application.py
